# Web app for generating images using Dall-e API

## Instructions

- Clone the repo
- create a .env file inside the server folder
  ```
    OPEN_API_KEY=
    MONGODB_URL=
    CLOUDINARY_CLOUD_NAME=
    CLOUDINARY_API_KEY=
    CLOUDINARY_API_SECRET=
    SOCKET_PORT=
    SECRET_KEY
  ```
- install the package  `npm i`
- start the app `npm start`
